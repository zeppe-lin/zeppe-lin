=head1 NAME

pkgman-search - search the packages sources directories for package
name

=head1 SYNOPSIS

B<pkgman> search [B<-v>|B<-vv>] [B<--path>] [B<--regex>] I<EXPR>

=head1 DESCRIPTION

Search the packages sources directories for I<EXPR> in packages' name.
It looks in all directories specified in F</etc/pkgman.conf> as
C<pkgsrcdir> directories.

=head1 OPTIONS

=over 8

=item B<-v>

Show version in the listing.

=item B<-vv>

Show version and description in the listing.

=item B<--path>

Show package source path in the listing.

=item B<--regex>

Interpret I<EXPR> as regular expression.

=item I<EXPR>

String or regular expression.

=back

=cut
